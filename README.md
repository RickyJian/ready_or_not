# Ready Or Not

簡單記帳 APP

## 功能

### 帳戶

帳戶可分為 `一般` 或 `特別` 帳戶：

* 一般：貨幣帳戶，用於一般貨幣交易，單位為各國 `幣值`
* 特別：非貨幣帳戶皆屬於特別帳戶，特別帳戶可設定 `匯率` 及 `單位`

基本屬性
* 名稱
* 初始額度：可為負數
* 備註
* 啟用
* 單位：若 `類別` 為 `一般` 單位為幣值，`特別` 則必填
* 匯率：若 `類別` 為 `特別` 匯率可選填
* 類別：新增後不可修改

> 停用帳戶不會在 `交易` 顯示
